# QAチェックリスト（実機）

## 基本
- [ ] 起動→クラッシュしない
- [ ] 新規個体作成ができる（複数回）
- [ ] アプリ再起動しても個体が消えない

## Focus（育成モード）
- [ ] 10/25/45/60が選べる
- [ ] 開始後UIロック（育成操作不可、表示のみ）
- [ ] 完走したら抽選が走る（当選/非当選どちらも確認）
- [ ] 中断したら抽選されない
- [ ] 中断してもGPは経過分だけ増えている（巻き戻らない）

## 突然変異
- [ ] 10分=1回、25=2回、45=3回、60=4回
- [ ] 当たりで打ち切り（最大1当選）
- [ ] 変異は5種のどれか
- [ ] 既に持っている変異を引いた場合、何も起きない（再抽選しない）
- [ ] 変異は永続（再起動しても残る）

## 成長
- [ ] 10分相当でGPが1増える
- [ ] 120GPで成熟扱いになる
